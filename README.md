## 这是什么？
 > 基于fis3和thinkjs搭建的后台管理系统（cms）
 > 数据库操作基于golang搭建的restful请求实现

## 运行（开发环境）

```js
npm install fis3 -g

cd fis-static

fis3 release -d ../ -wL

```

```js

cd pizzaManage

npm install

npm start

```
